# Custom Skills in Claude Code

Skills are reusable, packaged capabilities that extend Claude Code with custom behaviors and specialized tools. Think of them as plugins that transform Claude into a domain-specific assistant for any task you need.

## What You'll Learn

- How to create custom skills from scratch
- Understanding skill structure and configuration
- Handling arguments and parameters
- Working with available tools
- Best practices for building reliable skills
