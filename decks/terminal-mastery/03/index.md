# Essential Commands

Start with these fundamental commands.

| Command | Purpose | Example |
| ------- | ------- | ------- |
| `ls` | List files | `ls -la` |
| `cd` | Change directory | `cd ~/Documents` |
| `pwd` | Print working directory | `pwd` |
| `cat` | Display file contents | `cat file.txt` |
| `mkdir` | Create directory | `mkdir new-folder` |
| `rm` | Remove files/folders | `rm file.txt` |
| `cp` | Copy files | `cp src dest` |
| `mv` | Move/rename files | `mv old new` |

> **Tip**: Use `man <command>` to see the manual for any command.
