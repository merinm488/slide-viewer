# Navigation Shortcuts

Move through your filesystem like a pro.

**Directory Navigation**

```bash
cd ..      # Go up one level
cd ~       # Go to home directory
cd -       # Go to previous directory
```

**Tab Completion**

Type the beginning of a file/command and press `Tab` to autocomplete. This saves time and prevents typos.

**History Navigation**

```bash
# Press ↑ and ↓ to navigate command history
# Or search with Ctrl+R
(reverse-i-search)`': git
```

**Quick Jump**

```bash
# Create aliases for frequently used directories
alias proj="cd ~/Projects"
alias docs="cd ~/Documents"
```
