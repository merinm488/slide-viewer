# Terminal Mastery

A slide deck about mastering the command-line interface for developers.

## Slides Overview

| # | Title | Description |
|---|-------|-------------|
| 01 | Mastering the Terminal | Introduction to terminal proficiency |
| 02 | Why Learn the Terminal? | Benefits and use cases |
| 03 | Essential Commands | Fundamental command-line operations |
| 04 | Navigation Shortcuts | Efficient filesystem navigation |
| 05 | Power User Techniques | Piping, redirection, background jobs |
| 06 | Practice Resources | Interactive learning tools and references |

## Contents

This deck covers:
- Why the terminal is a developer superpower
- Essential commands (ls, cd, pwd, cat, mkdir, rm, cp, mv)
- Tab completion and history navigation
- Piping and chaining commands
- Output redirection
- Background jobs and process management
- Practice resources and games

## Format

Each slide is a markdown file (`index.md`) in a numbered folder. Images are stored alongside their slides.
