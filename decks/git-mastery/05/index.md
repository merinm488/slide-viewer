# Advanced Git Techniques

## Rebasing
```bash
git rebase main  # Clean history
```

## Cherry-picking
```bash
git cherry-pick abc123  # Apply specific commit
```

## Stashing
```bash
git stash push -m "WIP"
git stash pop
```
