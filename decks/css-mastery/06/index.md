# Modern CSS Features

## What's New in 2024
- **`:has()`** - Parent selector
- **Subgrid** - Nested grid alignment
- **Cascade Layers** - `@layer` for specificity control
- **Nesting** - Native CSS nesting!
- **Scroll-driven Animations**

```css
.card:has(.badge) {
  border: 2px solid gold;
}
```
